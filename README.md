# Expense Tracking System

A web-based Expense Tracker application built using **Java, Servlet, JSP, JDBC, and MySQL**.  
This project helps users manage their daily expenses efficiently with features like adding, viewing, updating, and deleting expenses.

---

## 🚀 Features
- User Registration & Login
- Add Expense
- View Expense List
- Update Expense
- Delete Expense
- Dashboard Page
- Logout Functionality

---

## 🛠️ Technologies Used
- Java
- Servlet
- JSP
- JDBC
- MySQL
- HTML, CSS
- Apache Tomcat Server

---

## 📂 Project Structure
- `src/main/java` → Java Servlets and Backend Code
- `src/main/webapp` → JSP pages, CSS files
- `WEB-INF/web.xml` → Servlet configuration
